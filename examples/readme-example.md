### say-hello

My super function.
Not thread-safe.

#### Example

```bash

```

#### Arguments

* **$1** (string): A value to print

#### Exit codes

* **0**: If successful.
* **1**: If an empty string passed.

#### Output on stdout

* Output 'Hello $1'.

#### Output on stderr

* Output 'Oups !' on error.
